Detailed Description

This class provides members to establish readings from the BME680 environmental sensor.

Readings include:

  • Ambient Temperature
  • Relative Humidity
  • Atmospheric Pressure

SW01_Altitude_Measurement.ino, SW01_Humidity_Measurement.ino, SW01_Pressure_Measurement.ino, and SW01_Temperature_Measurement.ino.
